# CONTRIBUTING

## Development

```bash
# fork
# git clone
# cd vite-plugin-checker
pnpm i
pnpm build
pnpm dev
```

## Release

To release a beta version

```bash
npm run release:beta
```

To release a official version

```bash
npm run release
```

Use `release:beta-dry-run` or `release:dry-run` to dry run release locally.

## Publish

```bash
npm run ci:publish
```
